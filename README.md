# Project for https://github.com/avito-tech/mi-trainee-task

## HOW TO START
```
make build
make up
```
or
```
docker-compose up --build 
```

## API
* root: `/`
* docs: `/docs`
* all keys: `/keys`
* generate key: `/generate`
* get secret: `/secrets/{secret}`
